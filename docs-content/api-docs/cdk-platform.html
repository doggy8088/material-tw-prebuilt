服務

Platform

本服務透過比較 userAgent 字串和檢查瀏覽器特定的全域屬性來檢測當前平台。

Service to detect the current platform by comparing the userAgent strings and checking browser-specific global properties.

屬性
名稱 描述

ANDROID: boolean

當前的平台是不是 Android 版。

Whether the current platform is Android.

BLINK: boolean

當前的渲染引擎是否為 Blink。

Whether the current rendering engine is Blink.

EDGE: boolean

當前的瀏覽器是否為 Microsoft Edge。

Whether the current browser is Microsoft Edge.

FIREFOX: boolean

當前的瀏覽器是否為 Firefox。

Whether the current browser is Firefox.

IOS: boolean

目前的平台是否為 Apple iOS。

Whether the current platform is Apple iOS.

SAFARI: boolean

當前的瀏覽器是否為 Safari。

Whether the current browser is Safari.

TRIDENT: boolean

當前的渲染引擎是否為 Microsoft Trident。

Whether the current rendering engine is Microsoft Trident.

WEBKIT: boolean

當前的渲染引擎是否為 WebKit。

Whether the current rendering engine is WebKit.

isBrowser: boolean

Angular 應用是否正在瀏覽器中渲染。

Whether the Angular application is being rendered in the browser.

檢查此瀏覽器使用的 RTL 捲動軸的型別。在撰寫本文時，Chrome 瀏覽器為 NORMAL（正常），Firefox 和 Safari 為 NEGATED（否），而 IE＆Edge 為 INVERTED（反向）。
