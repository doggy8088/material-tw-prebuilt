AutofillMonitor

一種可注入的服務，可以用來監控輸入框的自動填充狀態。根據以下部落格文章： https：//medium.com/@brunn/detecting-autofilled-fields-in-javascript-aed598d25da7

An injectable service that can be used to monitor the autofill state of an input. Based on the following blog post: https://medium.com/@brunn/detecting-autofilled-fields-in-javascript-aed598d25da7

指令

CdkAutofill

一個指令，用於監控輸入的自動填充狀態。

A directive that can be used to monitor the autofill state of an input.

選擇器： [cdkAutofill]
屬性
名稱 描述
@Output()

cdkAutofill: EventEmitter<AutofillEvent>

元素的自動填充狀態發生變化時觸發。

Emits when the autofill state of the element changes.

CdkTextareaAutosize

用於自動調整 textarea 大小以適應其內容的指令。

Directive to automatically resize a textarea to fit its content.

選擇器： textarea[cdkTextareaAutosize]
匯出為： cdkTextareaAutosize
屬性
名稱 描述

textarea 中的最大行數。

Maximum amount of rows in the textarea.
@Input('cdkAutosizeMinRows')

textarea 中的最小行數。

方法
reset

把 textarea 重置為原始大小

Resets the textarea to its original size
resizeToFitContent

調整 textarea 的大小以適應它的內容。

Resize the textarea to fit its content.
引數Parameters

force

boolean = false

是否強行重新計算高度。預設情況下，只有當自上一次呼叫後的值發生變化時，才會重新計算高度。

Whether to force a height recalculation. By default the height will be recalculated only if the value changed since the last call.
