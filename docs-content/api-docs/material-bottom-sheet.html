Angular Material bottom-sheet API 參考文件

import {MatBottomSheetModule} from '@angular/material/bottom-sheet';

服務

MatBottomSheet

用於觸發 Material Design 底部操作表的服務。

Service to trigger Material Design bottom sheets.

方法
dismiss

關掉當前可見的底部操作表。

Dismisses the currently-visible bottom sheet.
引數Parameters

R

要傳遞給底部操作表實例的資料。

Data to pass to the bottom sheet instance.
open

開啟一個包含指定元件的底部操作表。

Opens a bottom sheet containing the given component.
引數Parameters

component

ComponentType<T>

要裝入底部操作表的元件類別型。

Type of the component to load into the bottom sheet.

config?

Extra configuration options.

Reference to the newly-opened bottom sheet.

類

MatBottomSheetConfig

開啟底部操作表時使用的設定。

Configuration used when opening a bottom sheet.

屬性
名稱 描述

ariaLabel: string | null

指定給底部操作表元素的 Aria 標籤。

Aria label to assign to the bottom sheet element.

ariaModal: boolean

這個底部操作表是否是模態的。用於設定 aria-modal 屬性。

Whether this is a modal bottom sheet. Used to set the aria-modal attribute.

autoFocus: AutoFocusTarget | string | boolean

底部工作表開啟時應聚焦到的位置。。

Where the bottom sheet should focus on open.

backdropClass: string

背景板的自定義類。

Custom class for the backdrop.

closeOnNavigation: boolean

當用戶在歷史記錄中前進或後退時，底部操作表是否應該關閉。請注意，這通常不包括單擊某些連結（除非使用者正在使用 HashLocationStrategy ）。

Whether the bottom sheet should close when the user goes backwards/forwards in history. Note that this usually doesn't include clicking on links (unless the user is using the HashLocationStrategy).

data: D | null

注入到子元件中的資料。

Data being injected into the child component.

direction: Direction

底部操作表的文字佈局方向。

Text layout direction for the bottom sheet.

disableClose: boolean

使用者是否可以使用 escape 或單擊外部來關閉底部操作表。

Whether the user can use escape or clicking outside to close the bottom sheet.

hasBackdrop: boolean

底部操作表是否有背景板。

Whether the bottom sheet has a backdrop.

panelClass: string | string[]

要新增到底部操作表容器中的額外 CSS 類。

Extra CSS classes to be added to the bottom sheet container.

restoreFocus: boolean

關閉底部操作表時是否應該把焦點還給以前擁有焦點的元素。

Whether the bottom sheet should restore focus to the previously-focused element, after it's closed.

scrollStrategy: ScrollStrategy

用於底部操作表的捲動策略。

Scroll strategy to be used for the bottom sheet.

viewContainerRef: ViewContainerRef

用於放置底部操作表浮層的容器。

The view container to place the overlay for the bottom sheet into.

MatBottomSheetRef

引用來自底部操作表服務的底部操作表。

Reference to a bottom sheet dispatched from the bottom sheet service.

屬性
名稱 描述

componentRef: ComponentRef<T> | null

ComponentRef of the component opened into the bottom sheet. Will be null when the bottom sheet is opened using a TemplateRef.

disableClose: boolean | undefined

是否允許使用者關閉底部操作表。

Whether the user is allowed to close the bottom sheet.

instance: T

該組件的實例組成了底部操作表的內容。

Instance of the component making up the content of the bottom sheet.

常數

MAT_BOTTOM_SHEET_DEFAULT_OPTIONS

這個注入令牌可以用來指定底部操作表的預設選項。

Injection token that can be used to specify default bottom sheet options.

const MAT_BOTTOM_SHEET_DEFAULT_OPTIONS: InjectionToken<MatBottomSheetConfig<any>>;

MAT_BOTTOM_SHEET_DATA

這個注入令牌可以用來訪問那些傳入底部操作表的資料。

Injection token that can be used to access the data that was passed in to a bottom sheet.

const MAT_BOTTOM_SHEET_DATA: InjectionToken<any>;
