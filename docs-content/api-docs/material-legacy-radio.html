Angular Material legacy-radio API 參考文件

Import symbols from @angular/material/legacy-radio

指令

MatLegacyRadioGroup

一組單選按鈕。可以包含一個或多個 <mat-radio-button> 元素。

A group of radio buttons. May contain one or more <mat-radio-button> elements.

選擇器： mat-radio-group
匯出為： matRadioGroup
已棄用
屬性
名稱 描述
@Input()

color: ThemePalette

該組中所有單選按鈕的主題顏色。

Theme color for all of the radio buttons in the group.
@Input()

disabled: boolean

單選按鈕組是否已停用

Whether the radio group is disabled
@Input()

labelPosition: 'before' | 'after'

標籤是應該出現在單選按鈕之後還是之前。預設為 'after'

Whether the labels should appear after or before the radio-buttons. Defaults to 'after'
@Input()

name: string

單選按鈕組的名稱。該組中的所有單選按鈕都將使用此名稱。

Name of the radio button group. All radio buttons inside this group will use this name.
@Input()

required: boolean

單選按鈕組是否為必填項

Whether the radio group is required
@Input()

selected: T

當前選定的單選按鈕。如果設定為新的單選按鈕，則會更新該單選按鈕組的值以對應新選定的按鈕。

The currently selected radio button. If set to a new radio button, the radio group value will be updated to match the new selected button.
@Input()

value: any

單選按鈕組的值。如果有一個與對應值對應的單選按鈕，它應該等於選定單選按鈕的值。如果沒有這樣一個對應的單選按鈕，在加入了一個帶有對應值的新單選按鈕，該值仍然會生效。

Value for the radio-group. Should equal the value of the selected radio button if there is a corresponding radio button with a matching value. If there is not such a corresponding radio button, this value persists to be applied in case a new radio button is added with a matching value.
@Output()

change: EventEmitter<MatRadioChange>

本組的值發生變化時發出的事件。只有在值是因使用者與單選按鈕的互動而發生變化時才會發出變更事件（與 <input type-"radio"> 的行為相同）。

Event emitted when the group value changes. Change events are only emitted when the value changes due to user interaction with a radio button (the same behavior as <input type-"radio">).

MatLegacyRadioButton

Material Design 單選按鈕。通常放在 <mat-radio-group> 元素內。

A Material design radio-button. Typically placed inside of <mat-radio-group> elements.

選擇器： mat-radio-button
匯出為： matRadioButton
已棄用

名稱 描述
@Input('aria-describedby')

ariaDescribedby: string

'aria-describedby' 屬性是在該元素的標籤和欄位型別之後讀取的。

The 'aria-describedby' attribute is read after the element's label and field type.
@Input('aria-label')

ariaLabel: string

用來在底層的 input 元素上設定 'aria-label' 屬性。

Used to set the 'aria-label' attribute on the underlying input element.
@Input('aria-labelledby')

ariaLabelledby: string

'aria-labelledby' 屬性優先於該元素上的替換文字。

The 'aria-labelledby' attribute takes precedence as the element's text alternative.

Whether this radio button is checked.

Theme color of the radio button.

Whether ripples are disabled.

單選按鈕是否已停用。

Whether the radio button is disabled.

The unique ID for the radio button.
@Input()

labelPosition: 'before' | 'after'

標籤是應該出現在單選按鈕之後還是之前。預設為 'after'

Whether the label should appear after or before the radio button. Defaults to 'after'
@Input()

name: string

模擬 HTML 的 『name』 屬性，用於對單選按鈕進行分組以進行唯一選擇。

Analog to HTML 'name' attribute used to group radios for unique selection.

Whether the radio button is required.

@Output()

change: EventEmitter<MatRadioChange>

該單選按鈕的 checked 屬性發生變化時，會發出本事件。只會在值因使用者與單選按鈕的互動而發生變化時發出此變更事件（與 <input type-"radio"> 的行為相同）。

Event emitted when the checked state of this radio button changes. Change events are only emitted when the value changes due to user interaction with the radio button (the same behavior as <input type-"radio">).

Tabindex to which to fall back to if no value is set.
