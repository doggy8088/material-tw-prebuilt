Angular Material 入門指南

Getting Started with Angular Material

本指南講解瞭如何設定 Angular 專案以開始使用 Angular Material。它包含有關先決條件的訊息、如何安裝 Angular Material，以及在應用中顯示一個 Material 範例元件，以驗證你的設定。

This guide explains how to set up your Angular project to begin using Angular Material. It includes information on prerequisites, installing Angular Material, and optionally displaying a sample Material component in your application to verify your setup.

Angular 資源

Angular Resources

如果你是 Angular 新手，或從一個新的 Angular 應用開始，參見 Angular 的完整入門指南和設定你的環境。

If you are new to Angular or getting started with a new Angular application, see Angular's full Getting Started Guide and Setting up your environment.

對於已有的應用，請按照以下步驟開始使用 Angular Material。

For existing applications, follow the steps below to begin using Angular Material.

安裝 Angular Material

Install Angular Material

使用 Angular CLI 的安裝原理圖透過執行以下指令來設定你的 Angular Material 專案：

Use the Angular CLI's installation schematic to set up your Angular Material project by running the following command:

ng add @angular/material

該 ng add 指令將安裝 Angular Material、元件開發工具包（CDK）、Angular 動畫，並詢問下列問題，以確定要包括哪些功能：

The ng add command will install Angular Material, the Component Dev Kit (CDK), Angular Animations and ask you the following questions to determine which features to include:

  1. 選擇一個預置的主題名或為自定義主題選擇 『custom』：

    Choose a prebuilt theme name, or "custom" for a custom theme:

    你可以選擇一個預置的 Material Design 主題或建立一個可擴充套件的自定義主題。

    You can choose from prebuilt material design themes or set up an extensible custom theme.

  2. 設定全域 Angular Material 排版樣式：

    Set up global Angular Material typography styles:

    是否為應用程式套用全域排版樣式。

    Whether to apply the global typography styles to your application.

  3. 為 Angular Material 設定瀏覽器動畫：

    Set up browser animations for Angular Material:

    把 BrowserAnimationsModule匯入到應用中，可以啟用 Angular 的動畫體系。拒絕它會使 Angular Material 中的動畫失效。

    Importing the BrowserAnimationsModule into your application enables Angular's animation system. Declining this will disable most of Angular Material's animations.

ng add 指令還將執行以下動作：

The ng add command will additionally perform the following actions:

  • 把專案依賴加入到 package.json 中

    Add project dependencies to package.json

  • 把 Roboto 字型新增到你的 index.html 中

    Add the Roboto font to your index.html

  • 把 Material Design 的圖示字型新增到你 index.html 中

    Add the Material Design icon font to your index.html

  • 新增一些全域 CSS 樣式：

    Add a few global CSS styles to:

    • 去掉 body 的邊距

      Remove margins from body

    • 為 html 和 body 設定 height: 100%

      Set height: 100% on html and body

    • 把 Roboto 設定為預設的應用字型

      Set Roboto as the default application font

完工！ Angular Material 已經設定好應用於你的應用中了。

You're done! Angular Material is now configured to be used in your application.

顯示一個元件

Display a component

讓我們在應用中顯示一個滑區塊開關元件，來驗證一切正常。

Let's display a slide toggle component in your app and verify that everything works.

你需要透過把以下程式碼新增到 app.module.ts 檔案中來匯入 MatSlideToggleModule。

You need to import the MatSlideToggleModule that you want to display by adding the following lines to your app.module.ts file.

import { MatSlideToggleModule } from '@angular/material/slide-toggle';

@NgModule ({
  imports: [
    MatSlideToggleModule,
  ]
})
class AppModule {}

把 <mat-slide-toggle> 標籤新增到 app.component.html，就像這樣：

Add the <mat-slide-toggle> tag to the app.component.html like so:

<mat-slide-toggle>Toggle me!</mat-slide-toggle>

執行本地開發伺服器：

Run your local dev server:

ng serve

讓瀏覽器訪問 http://localhost:4200

Then point your browser to http://localhost:4200

你會在頁面上看到這個 Material 滑區塊開關元件。

You should see the Material slide toggle component on the page.

除了安裝原理圖之外，Angular Material 還帶有另外一些原理圖（如導向、表格、地址表單等），可用於方便地在應用中生成預置的元件。

In addition to the installation schematic, Angular Material comes with several other schematics (like nav, table, address-form, etc.) that can be used to easily generate pre-built components in your application.
