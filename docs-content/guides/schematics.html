安裝與程式碼生成

Installation and Code Generation

Angular Material 帶了一份供 Angular CLI 使用的原理圖（schematic），可以讓建立 Material 應用更簡單些。

Angular Material comes packaged with Angular CLI schematics to make creating Material applications easier.

安裝原理圖

Install Schematics

Angular Material 附帶了一些原理圖，一旦你安裝了這個 npm 套件，就可以透過 Angular CLI 來使用這些原理圖了。

Schematics are included with both @angular/cdk and @angular/material. Once you install the npm packages, they will be available through the Angular CLI.

執行下列指令可以自動為你的專案安裝 Angular Material、元件開發工具 (CDK) 和 Angular 動畫庫，並執行程式碼原理圖。

Using the command below will install Angular Material, the Component Dev Kit (CDK), and Angular Animations in your project. Then it will run the installation schematic.

ng add @angular/material

如果你只想安裝 @angular/cdk，也可以使用元件開發工具的專用程式碼原理圖

In case you just want to install the @angular/cdk, there are also schematics for the Component Dev Kit

ng add @angular/cdk

Angular Material 的 ng add 原理圖會幫助你的 Angular CLI 專案設定 Angular Material。執行 ng add 將會：

The Angular Material ng add schematic helps you set up an Angular CLI project that uses Material. Running ng add will:

  • 確保 package.json 中具有專案所需的依賴

    Ensure project dependencies are placed in package.json

  • 為應用啟用 BrowserAnimationsModule 模組

    Enable the BrowserAnimationsModule in your app module

  • 新增一個預定義的主題或一個自定義主題

    Add either a prebuilt theme or a custom theme

  • 為 index.html 新增 Roboto 字型

    Add Roboto fonts to your index.html

  • 在 index.html 中新增 Material 圖示字型

    Add the Material Icon font to your index.html

  • 新增下列全域樣式

    Add global styles to

    • 移除 body 的邊距（margin）

      Remove margins from body

    • 為 html 和 body 設定 height: 100%

      Set height: 100% on html and body

    • 把 Roboto 設定為應用的預設字型

      Make Roboto the default font of your app

元件原理圖

Component schematics

除了用於安裝的 schematic 之外，Angular Material 還帶有多個原理圖，用於輕鬆生成 Material Design 元件：

In addition to the installation schematic, Angular Material comes with multiple schematics that can be used to easily generate Material Design components:

名稱Name 說明Description
address-form 一個表單組，它使用 Material Design 的多個表單控制元件接收一個送貨地址Component with a form group that uses Material Design form controls to prompt for a shipping address
navigation 建立一個帶有回應式 Material Design 側邊欄元件和一個用於顯示應用名稱的工具欄元件Creates a component with a responsive Material Design sidenav and a toolbar for showing the app name
dashboard 帶有多個 Material Design 卡片和選單的網格式佈局元件Component with multiple Material Design cards and menus which are aligned in a grid layout
table 生成一個帶有 Material Design 並支援排序和分頁的資料表元件Generates a component with a Material Design data table that supports sorting and pagination
tree 該組件以 <mat-tree> 元件來展示一個可互動的資料夾式結構Component that interactively visualizes a nested folder structure by using the <mat-tree> component

此外，Angular CDK 也給出了一些元件的原理圖：

Additionally, the Angular CDK also comes with a collection of component schematics:

名稱Name 說明Description
drag-drop 該組件使用 @angular/cdk/drag-drop 指令來建立一個可互動的 to-do 清單Component that uses the @angular/cdk/drag-drop directives for creating an interactive to-do list

地址表單原理圖

Address form schematic

執行 address-form 原理圖會生成一個新的 Angular 元件，它可用於快速開始一個 Material Design 表單組，其中包括：

Running the address-form schematic generates a new Angular component that can be used to get started with a Material Design form group consisting of:

  • 一些 Material Design 表單欄位

    Material Design form fields

  • 一些 Material Design 單選控制元件

    Material Design radio controls

  • 一些 Material Design 按鈕

    Material Design buttons

ng generate @angular/material:address-form <component-name>

導向原理圖

Navigation schematic

navigation 原理圖將會建立一個包括應用名的工具欄和一個能自適應 Material 的斷點的側邊欄。

The navigation schematic will create a new component that includes a toolbar with the app name, and a responsive side nav based on Material breakpoints.

ng generate @angular/material:navigation <component-name>

表格原理圖

Table schematic

表格原理圖將建立一個元件，它可以渲染出一個預置了可排序、可分頁資料來源的 Angular Material <table>。

The table schematic will create a component that renders an Angular Material <table> which has been pre-configured with a datasource for sorting and pagination.

ng generate @angular/material:table <component-name>

儀表盤原理圖

Dashboard schematic

dashboard 原理圖將建立一個新元件，它包含一個由 Material Design 卡片組成的動態網格清單。

The dashboard schematic will create a new component that contains a dynamic grid list of Material Design cards.

ng generate @angular/material:dashboard <component-name>

樹原理圖

Tree schematic

tree 原理圖可用於快速生成一個 Angular 元件，它使用 Angular Material 的 <mat-tree> 元件來展示一個巢狀的資料夾式結構。

The tree schematic can be used to quickly generate an Angular component that uses the Angular Material <mat-tree> component to visualize a nested folder structure.

ng generate @angular/material:tree <component-name>

拖放原理圖

Drag and Drop schematic

drag-drop 原理圖是由 @angular/cdk 提供的，它可用來生成帶有 CDK 拖放指令的元件。

The drag-drop schematic is provided by the @angular/cdk and can be used to generate a component that uses the CDK drag and drop directives.

ng generate @angular/cdk:drag-drop <component-name>
