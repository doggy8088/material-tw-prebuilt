portals 套件提供了一個靈活的佈局體系，可以把動態內容渲染到應用中。

The portals package provides a flexible system for rendering dynamic content into an application.

傳送點

Portals

Portal 是一小塊 UI，可以被動態渲染到頁面上的空白插槽（slot）中。

A Portal is a piece of UI that can be dynamically rendered to an open slot on the page.

這『一小塊 UI』既可以是 Component、TemplateRef 或 DOM 元素，而『空白的插槽』是指 PortalOutlet。

The "piece of UI" can be either a Component, a TemplateRef or a DOM element and the "open slot" is a PortalOutlet.

各種 Portal 和 PortalOutlet 是建立在其它概念（比如浮層）之上的底層構造塊。

Portals and PortalOutlets are low-level building blocks that other concepts, such as overlays, are built upon.

Portal<T>
方法Method 說明Description
attach(PortalOutlet): T 把此傳送點附著到宿主上。Attaches the portal to a host.
detach(): void 把此傳送點從宿主上拆除。Detaches the portal from its host.
isAttached: boolean 此傳送點是否已附著上。Whether the portal is attached.
PortalOutlet
方法Method 說明Description
attach(Portal): any 把指定的傳送點新增到此宿主上。Attaches a portal to the host.
detach(): any 把指定的傳送點從此宿主上拆除。Detaches the portal from the host.
dispose(): void 永久銷燬此宿主。Permanently dispose the host.
hasAttached: boolean 判斷是否有傳送點附著在此宿主上。Whether a portal is attached to the host.

傳送點實戰

Portals in practice

CdkPortal

用於從某個 <ng-template> 獲取傳送點。CdkPortal 是一個 Portal。

Used to get a portal from an <ng-template>. CdkPortal is a Portal.

用法：

Usage:

<ng-template cdkPortal>
  <p>The content of this template is captured by the portal.</p>
</ng-template>

<!-- OR -->

<!-- This result here is identical to the syntax above -->
<p *cdkPortal>
  The content of this template is captured by the portal.
</p>

元件可以使用 @ViewChild 或 @ViewChildren 來獲取對 CdkPortal 的引用。

A component can use @ViewChild or @ViewChildren to get a reference to a CdkPortal.

ComponentPortal

用於從某個元件類別建立傳送點。

Used to create a portal from a component type.

用法：

Usage:

ngAfterViewInit() {
  this.userSettingsPortal = new ComponentPortal(UserSettingsComponent);
}
TemplatePortal

你可以從 <ng-template> TemplatePortal。TemplatePortal 允許你在一個樣板中獲取 Angular 的內容，並把它渲染到其他地方。

You can create a TemplatePortal from an <ng-template>. TemplatePortal allows you to take Angular content within one template and render it somewhere else.

用法：

Usage:

<ng-template #templatePortalContent>Some content here</ng-template>

@ViewChild('templatePortalContent') templatePortalContent: TemplateRef<unknown>;

ngAfterViewInit() {
  this.templatePortal = new TemplatePortal(
    this.templatePortalContent,
    this._viewContainerRef
  );
}
DomPortal

你可以從任何的 native DOM 元素 DomPortal DomPortal 允許你把任意 DOM 內容帶到其它地方。DomPortal如移動內容，因此，如果經由移動，角元件設有像繫結或指令可能不再更新 DomPortal。

You can create a DomPortal from any native DOM element. DomPortal allows you to take any arbitrary DOM content and render it somewhere else. DomPortal moves content as is, so elements with Angular features like bindings or directives may no longer update if moved via DomPortal.

用法：

Usage:

<div #domPortalContent>Some content here</div>

@ViewChild('domPortalContent') domPortalContent: ElementRef<HTMLElement>;
ngAfterViewInit() {
  this.domPortal = new DomPortal(this.domPortalContent);
}
CdkPortalOutlet

用於向樣板中新增一個傳送點出口（Portal Outlet）。CdkPortalOutlet 是一個 PortalOutlet。

Used to add a portal outlet to a template. CdkPortalOutlet is a PortalOutlet.

用法：

Usage:

<!-- Attaches the `userSettingsPortal` from the previous example. -->
<ng-template [cdkPortalOutlet]="userSettingsPortal"></ng-template>
