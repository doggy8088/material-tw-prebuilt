<mat-progress-spinner> 和 <mat-spinner> 是一個表示進度和活動的圓形指示符。

<mat-progress-spinner> and <mat-spinner> are a circular indicators of progress and activity.

進度模式

Progress mode

進度圈支援兩種模式：確定（"determinate"）和待定（"indeterminate"）。

The progress-spinner supports two modes, "determinate" and "indeterminate". The <mat-spinner> component is an alias for <mat-progress-spinner mode="indeterminate">.

模式Mode 說明Description
determinate 標準的進度指示符，填充 0% 到 100%Standard progress indicator, fills from 0% to 100%
indeterminate 表示正在發生某些事情，卻沒有傳達離散的進度Indicates that something is happening without conveying a discrete progress

預設模式是確定的（"determinate"）。在該模式下，進度要透過 value 屬性進行設定，它可以是 0 到 100 之間的整數。

The default mode is "determinate". In this mode, the progress is set via the value property, which can be a whole number between 0 and 100.

在 "indeterminate" 模式下，會忽略 value 屬性。

In "indeterminate" mode, the value property is ignored.

主題

Theming

進度圈的顏色可以透過 color 屬性進行修改。預設情況下，進度圈使用主題的主色調（primary）。 它可以修改為 'accent' 或 'warn'。

The color of a progress-spinner can be changed by using the color property. By default, progress-spinners use the theme's primary color. This can be changed to 'accent' or 'warn'.

無障礙性

Accessibility

MatProgressSpinner 實現了 ARIA role="progressbar" 模式。預設情況下，微調器將 aria-valuemin 設定為 0 ，將 aria-valuemax 為 100 。儘量不要更改這些值，因為這可能會導致與某些輔助技術不相容。

MatLegacyProgressSpinner implements the ARIA role="progressbar" pattern. By default, the spinner sets aria-valuemin to 0 and aria-valuemax to 100. Avoid changing these values, as this may cause incompatibility with some assistive technology.

始終透過 aria-label 或 aria-labelledby 為每個微調器提供無障礙標籤。

Always provide an accessible label via aria-label or aria-labelledby for each spinner.
