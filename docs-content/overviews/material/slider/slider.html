像 <input type="range"> 一樣，<mat-slider> 允許透過滑鼠、觸控或鍵盤來從範圍中選擇一個值。

<mat-slider> allows for the selection of a value from a range via mouse, touch, or keyboard, similar to <input type="range">.

注意：該組件的滑動行為要求本頁面中載入過 HammerJS 函式庫。

Note: the sliding behavior for this component requires that HammerJS is loaded on the page.

選擇一個值

Selecting a value

預設情況下，滑桿（slider）的最小值是 0，最大值是 100，滑區塊（thumb）移動時的增量是 1。 這些值可以透過設定 min、max 和 step 屬性進行修改。除非另行指定，否則其初始值就等於最小值。

By default the minimum value of the slider is 0, the maximum value is 100, and the thumb moves in increments of 1. These values can be changed by setting the min, max, and step attributes respectively. The initial value is set to the minimum value unless otherwise specified.

<mat-slider min="1" max="5" step="0.5" value="1.5">
  <input matSliderThumb>
</mat-slider>

選擇範圍

Selecting a range

還可以指定 invert 屬性，來翻轉坐標軸（滑區塊會沿著它移動）。 翻轉後的水平滑桿，其最小值在右側，而最大值在左側；翻轉後的垂直滑桿，其最小值在上部，而最大值在底部。

A <mat-slider> can be converted into a range slider by projecting both a matStartThumb and a matEndThumb into it. Each of the thumbs has an independent value, but they won't be allowed to overlap and they're still bound by the same min and max from the slider.

<mat-slider>
  <input matSliderStartThumb>
  <input matSliderEndThumb>
</mat-slider>

滑區塊標籤

Thumb label

Material Design 規範建議只為那些用來顯示離散值（比如 1~5 的打分值）的滑桿使用 thumbLabel 屬性（同時設定 tickInterval="1"）。

By default, the exact selected value of a slider is not visible to the user. However, this value can be added to the thumb by adding the discrete attribute.

<mat-slider discrete>
  <input matSliderThumb>
</mat-slider>

格式化滑區塊標籤

Formatting the thumb label

預設情況下，滑區塊標籤上的值會始終和模型的值相同，不過它可能太大，而沒辦法放進標籤中。 如果你要控制要顯示的值，你可以使用輸入屬性 displayWith 來實現。

By default, the value in the slider's thumb label will be the same as the model value, however this may end up being too large to fit into the label. If you want to control the value that is being displayed, you can do so using the displayWith input.

刻度標記

Tick marks

預設情況下，滑桿不會在滑區塊的導軌（track）上顯示刻度標記，不過可以使用 tickInterval 屬性來啟用它。tickInterval 的值應該是一個表示刻度之間步長的數字。比如 tickInterval 為 3 而 step 為 4 時，將會每隔 3 步畫出一個標記，每 12 個值重複一輪。

By default, sliders do not show tick marks along the thumb track. This can be enabled using the showTickMarks attribute.

<mat-slider showTickMarks>
  <input matSliderThumb>
</mat-slider>

Material Design 規範 建議只為那些用來顯示離散值（比如 1~5 的打分值）的滑桿設定 tickInterval 屬性（同時把 thumbLabel 屬性設定為 1）。

鍵盤互動

Keyboard interaction

滑桿具有下列鍵盤繫結：

The slider has the following keyboard bindings:

按鍵Key 操作Action
右方向鍵Right arrow 滑桿的值增加一步（在 RTL 下則減小）。Increment the slider value by one step (decrements in RTL).
上方向鍵Up arrow 滑桿的值增加一步。Increment the slider value by one step.
左方向鍵Left arrow 滑桿的值減小一步（在 RTL 下則增加）。Decrement the slider value by one step (increments in RTL).
下方向鍵Down arrow 滑桿的值減小一步。Decrement the slider value by one step.
上翻頁Page up 滑桿的值增加 10 步。Increment the slider value by 10 steps.
下翻頁Page down 滑桿的值減小 10 步。Decrement the slider value by 10 steps.
End 設定為可能的最大值。Set the value to the maximum possible.
Home 設定為可能的最小值。Set the value to the minimum possible.

無障礙性

Accessibility

MatSlider 實現了 ARIA role="slider" 模式，它會處理鍵盤輸入和焦點管理。始終要透過 aria-label 或 aria-labelledby 為每個滑區塊提供無障礙標籤。

MatSlider uses an internal <input type="range"> to provide an accessible experience. The input receives focus and it can be labelled using aria-label or aria-labelledby.
